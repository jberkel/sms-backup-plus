SMS Backup+ 1.5.5-BETA

Author: Jan Berkel (and many others, see below)

Automatically backs up your SMS/MMS and call logs to Gmail or any other IMAP server. Can also restore SMS / call logs after they have been backed up.

Information

Please visit the SMS Backup+ github page for more information. Also make sure to check the FAQ before sending me an email.

New in this release:

  • Fix call logs not getting backed up
  • Fix battery drain

Recent release history

  • 1.5.4 Various bugfixes (MMS not getting backed up, extra labels created)
  • 1.5.3 Bugfixes, Korean / Galician translation
  • 1.5.2 Fix notifications getting displayed while app is running in background

See changes on github for the full changelog.
